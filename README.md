## Create .env file
Create a .env file to store your database credentials

```
DB_HOST = localhost
DB_NAME = usermanagement_tut
DB_USER = root
DB_PASS = password
```

## Installation
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
